## Приложение для учёта студентов

Приложение для администрирования списка студентов.

- PWA
- HTTP-Сессии
- Material UI
- StyledComponents
- Регистрация, авторизация
- Подтверждение по аккаунта по почте
- Загрузка файлов на сервер
- Сброс пароля
- Custom Hooks
- Unit Testing
- E2E Testing
- И многие другие фичи...
